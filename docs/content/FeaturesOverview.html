Features list

This is a brief but detailed description of the main features offered by "Mp3 Studio".

  • File warning options
    • be warned if the file name is longer than 64 characters (usually you can't put in a CD a file whose name is longer than 64 characters!)
    • be warned if a file is read-only
    • be warned if a file with no mp3 extension is an mp3 file (sometimes it happens if your friends didn't use this program to organize their mp3s ...)
    •  
  • Tagging features
    • decide what kind of tags you want to write (version 1, version 2, both ...)
    • write tags basing on file name format
    • rename files by tag
    • mass tag operations (remove/clear/copy operations on some/all tag fields of version 1/version 2 tags)
    • edit tag operations (manually write tag fields)
    • support for reading/writing in editing/mass tag mode for more than 30 fields of tag version 2 that were not present in tag version 1
  •   Rename features

    You can use all these features to every kind of file, not only to mp3 files!

    • replace strings in file names
    • insert strings in file names
    • insert consecutive numbers in file names (useful if you like to put numbers in your mp3 file names)
    • powerful multi-replace feature (to change file name formats if you don't like it, for example "artist - track - title.mp3" to "track - artist - title.mp3").
    • case sensitivity selection for multi-replace and replace operations
    • uppercase, lowercase, large case operations
  • List features
    • create winamp (or other mp3 players) lists selecting all the directories in which you are interested to, with recursion, path-type and list-mode options.
  •   Database features
    • create a database with the most interesting mp3 fields (artist, title, album, year, genre, comment, track number, song length, file name, mp3 rate, mp3 sample rate)
    • add a user field to the produced list to identify the CD number
    • retrieve the album/artist/title fields first from tags and then from the file name or the folder name (if the tags are empty) or viceversa.
    • order the output list by artist and/or by name and/or by title and/or by file name
     
  • Organizer feature
    • reorganize your mp3 directories basing the reorganization on the genre field, the year field ...
  • Doubles feature
    • after having created your database, search for duplicated mp3 songs inside it!!
    • delete the duplicated songs by pressing a button in the duplicated song list
    • save to disk the list of duplicated songs
    • perform the search on multiple databases or directories!
  • CD optimizer feature
    • select the directories you want to put in a CD and let "Mp3 Studio" create the directorys ready to be put on a CD, optimizing CD space occupation!
     
  • Database features
    • load your database file and perform reordering and filtering operations to search songs
    • export your database file into html format
  • Mp3 features
    • reads all mp3 files characteristics, and gives the real song length also for Vbr files (Variable Bit Rate files), even better than what winamp does!
    • "Mp3 Studio" remembers all the configurations, the windows dimensions and positions.
