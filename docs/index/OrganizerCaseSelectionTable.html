Organizer Window

This window lets you reorganize your mp3 files in directories and subdirectories with names that are based on the tag values.

Case selection table

The case selection table is placed on the center-right side of the tag by name window.

On the first column there are the fields on which you can apply a case conversion. On the second column you can select the case conversion that has to be applied to the selected field.

The case conversion is applied to all the tags that are read on the selected fields, so that you can create your directories with the case format you like best. The effects of a case conversion can be better explained with some examples. Suppose that you have the following file name:

"beaUTiFul DAY"

If you select "Large Case" you will have the following result:

"Beautiful Day"

If you select "Lower Case" you will have the following result:

"beautiful day"

If you select "Upper Case" you will have the following result:

"BEAUTIFUL DAY"

If you select "First Capitalized" you will have the following result:

"BeaUTiFul DAY"

If you select nothing, no letter is touched and the word is left as is:

"beaUTiFul DAY"
