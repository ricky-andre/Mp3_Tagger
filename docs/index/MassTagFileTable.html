Mass Tag Window

The "mass tag window" is used to perform the same operation on all the selected files. This operation can be tag v1/v2 clear/remove/write. You can also write the tag version 2 advanced fields, all the new fields introduced by tag version 2 standard.

File table

Contains all the files that have been loaded from the main window. All most important tag version 1 and 2 fields are showed in this table, and are updated after a try or execute operation. Warning messages are printed in the warning panel.
