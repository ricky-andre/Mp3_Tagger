Doubles Window

This window gives the way to find out if you have duplicated songs inside your database. You can search inside a database file created with "Mp3 Studio" or inside the directories you have selected in some way.

Other options

Select the recurse subdirectories if you want all the selected directories to be scanned with all the included subdirectories.

You can also decide if you want the "advanced tag search" to be applied when retrieving tag values. The "advanced tag search" window is opened from the database creator window. Read carefully the help related to this milestone before applying this options.
