Tag by Name Window

The "tag by name window" is used to write tags on selected files using the file name value. You can perform case conversions, and edit the "expected file name format" to face particular cases.

Read the "configuration examples" in the content window to fully understand how this window can be used in all its features.

Try button

The "try" button is useful to understand what kind of fields will be written to the selected files. I always reccomend you to press this button before the execute one. On the file table will appear all the fields that will be written and the warnings will appear in the warning table.
