Mass Tag Window

The "mass tag window" is used to perform the same operation on all the selected files. This operation can be tag v1/v2 clear/remove/write. You can also write the tag version 2 advanced fields, all the new fields introduced by tag version 2 standard.

Warning Panel

All the warning messages that have to be showed during or after a try or execute operation are showed in the warning panel. The red sphere means that something was wrong, the green sphere means that everything has been performed with success!
