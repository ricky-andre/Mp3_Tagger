Rename By Tag Window

The "rename by tag window" is used to rename files using their tag values.

Fields Selection Table

On the top of the window, you can select the fields you want to use to rename files. The selected fields are put one after the other on the table, but you can change the order of them by selecting the fields and clicking on the "move up" or "move down" buttons: the output file name format is updated immediately.

To remove one or more fields, select them and press the "delete field" button.

You can also type-in the field name to insert a constant string.
