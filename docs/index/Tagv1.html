Tag v1

128 bytes of data written at the end of the file to store some useful information about the song. These are the stored fields and their range in the 128 bytes:
  • 'TAG':Indicates that the tag version 1 is present. Byte range: 0-2.
  • title: song's title, it can be long up to 30 characters. Byte range: 3-32.
  • artist: song's artist name, it can be long up to 30 characters. Byte range: 33-62.
  • album: song's album name, it can be long up to 30 characters. Byte range: 63-92.
  • year: song's release year, it can be long up to 4 characters. Byte range: 93-96.
  • comment: comment field, it can be long up to 29 characters. Byte range: 97-125.
  • track: song's track number, one byte representing the song number. Byte: 126.
  • genre: song's genre, represented with a one-byte code number. Byte: 127.
