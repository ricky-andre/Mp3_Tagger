Mass Tag Window

The "mass tag window" is used to perform the same operation on all the selected files. This operation can be tag v1/v2 clear/remove/write. You can also write the tag version 2 advanced fields, all the new fields introduced by tag version 2 standard.

Try button

The "try" button is useful to understand what kind of fields will be written/cleaned/removed from the selected files. I always reccomend you to press this button before the execute one. The file table will be updated to show what kind of fields values you will have by pressing the "execute" button.
